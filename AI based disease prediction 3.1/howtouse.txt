================================================================
        HOW TO USE — AI DISEASE PREDICTION SYSTEM
================================================================

STEP 1: INSTALL DEPENDENCIES
------------------------------
Make sure Python 3.8+ is installed. Then run:

    pip install -r requirements.txt

STEP 2: TRAIN THE ML MODEL
----------------------------
Run this once before starting the app:

    python model.py

This trains Random Forest, Naive Bayes, and Gradient Boosting
models on the dataset and saves the best one to models/model.pkl

STEP 3: SET UP OLLAMA (for AI Chatbot)
----------------------------------------
1. Download Ollama from: https://ollama.com/download
2. Install it on your system
3. Open a terminal and run:

    ollama run tinyllama

   (Keep this terminal open while using the app)

   Note: If ollama is not in PATH on Windows, use full path:
   "C:\Users\<YourName>\AppData\Local\Programs\Ollama\ollama.exe" run tinyllama

STEP 4: RUN THE APP
---------------------
In a separate terminal, run:

    python app.py

Then open your browser and go to:

    http://localhost:5000

STEP 5: USING THE APP
-----------------------
1. SIGNUP / LOGIN
   - First time? Click "Create New Account" on the login page
   - Login with your email and password
   - An intro animation (a1.mp4) plays after login

2. WELCOME PAGE
   - Shows the main background image
   - Click "Select your symptoms" to proceed
   - An intro animation (a2.mp4) plays before the symptoms page

3. SYMPTOMS PAGE
   - Search for symptoms using the search box
   - Tick all symptoms you are experiencing
   - Click "Predict Disease" to get your result

4. RESULT PAGE (PRESCRIPTION)
   - Shows predicted disease with confidence %
   - Top 3 differential diagnoses with confidence bars
   - Disease info: description, causes, precautions, medicines
   - Print the prescription using the Print button

5. DR. ATLAS CHATBOT
   - Click the 🤖 button (bottom right of symptoms page)
   - Describe your symptoms in plain language
   - Dr. Atlas gives structured medical advice
   - Click "Select Symptoms for Prescription" to go to the form

6. HISTORY
   - View all your past predictions with symptoms and dates

7. DASHBOARD
   - See total symptoms, diseases, models trained
   - View most predicted diseases (doughnut chart)
   - Model accuracy comparison (bar chart)

8. CONTACT US
   - Click "Contact Us" in the footer
   - Click "✍️ Write to Us" to send feedback
   - Messages are saved to feedback.txt

NOTES
------
- feedback.txt is created automatically when first feedback is sent
- users.db is created automatically on first run
- models/model.pkl must exist before running app.py
- All background images must be in the static/ folder
- a1.mp4 and a2.mp4 must be in the static/ folder for animations

TROUBLESHOOTING
----------------
- "model not found" error → Run python model.py first
- Chatbot says "Ollama not running" → Start ollama in a terminal
- Login fails after password change → Old plain-text passwords
  won't work after the hashing upgrade. Create a new account.
- Port already in use → Change port in app.py:
  app.run(debug=True, port=5001)
