        AI DISEASE PREDICTION SYSTEM — MediBot AI Clinic
================================================================

PROJECT OVERVIEW
----------------
An AI-powered full-stack web application that predicts diseases
based on user-selected symptoms using machine learning models,
combined with an AI chatbot (Dr. Atlas) powered by Ollama.

TECH STACK
----------
Backend  : Python, Flask
ML       : scikit-learn (Random Forest, Naive Bayes, Gradient Boosting)
AI Chat  : Ollama (tinyllama / llama3)
Database : SQLite (users.db)
Frontend : HTML, CSS (custom), JavaScript
Auth     : Werkzeug password hashing

KEY FEATURES
------------
- User authentication (signup / login) with hashed passwords
- Symptom selection with search and keyboard navigation
- ML-based disease prediction with top 3 differential diagnoses
- Prescription-style result page with confidence bar
- AI chatbot "Dr. Atlas" for symptom queries
- AI + ML integration in chat (extracts symptoms, runs ML model)
- Prediction history per user
- Dashboard with real-time stats and charts
- Contact Us page with feedback saved to feedback.txt
- Animated intro videos (a1.mp4, a2.mp4) on login and symptoms
- Background images per page (main.jpg, login.jpg, symptoms.jpg, chat.png, contact.jpg)
- Print-ready prescription page

PROJECT STRUCTURE
-----------------

  symptoms.jpg      — Symptoms page background
  chat.png          — Chatbot open background
  contact.jpg       — Contact page background
  a1.mp4            — Login intro animation
  a2.mp4            — Symptoms intro animation

templates/
  index.html        — Welcome / landing page
  symptoms.html     — Symptom selection + Dr. Atlas chatbot
  result.html       — Prescription result page
  login.html        — Login page
  signup.html       — Signup page
  history.html      — Prediction history
  dashboard.html    — Stats and charts
  contact.html      — Contact / feedback page
